Nuestra Wiki

Esta Wiki, creada por Alex Lara, Aritz Perez, Gotzon Otazua y Alex Mendiluce, tiene como objetivo explicar paso a paso y de manera sencilla la instalación de Owncloud, Prestashop y Wordpress en nuestro servidor.


A lo largo de cuatro semanas hemos recopilado informacion sobre los programas mencionados anteriormente, para poder preparar una documentación comprensible para todo el mundo. Tras una breve explicación de dicho programa, explicaremos los pasos a seguir para instalarlos, con la ayuda de varias imágenes tomadas en el proceso.


En la creación de esta wiki se han utilizado: HTML, CSS, Proxmox, GIT, GITHUB.


Apache, PHP y MySQL

Para la instalación de Apache, MySQL, PHP hemos decidido hacerlo de una manera simple, hemos utilizado un paquete complejo de instalación, dicho paquete de instalación es como si fuera un programa a descargar pero lo único que tiene es todos los comandos de instalación de los paquetes deseados.


Para realizar la instalación unicamente habría que ejecutar el comando que ejecuta el paquete, una vez finalice tendríamos ya instalados tanto el Apache, MySQL y PHP, aparte creara una tarea para cuando iniciemos el equipo inicie los servicios automáticamente.

Owncloud

Para la instalación de OwnCloud primero tenemos que descargarlo desde la Web oficial o mediante la consola de comandos de linux

Una vez descargado en el servidor vamos a extraerlo y le daremos permisos de lectura y escritura a su carpeta para la instalación sin problemas (en consola, " sudo chmod +rw 'nombre de carpeta'").

Comando de instalación

Una vez finalizado el anterior proceso accederemos a la dirección IP de nuestro servidor, ahi accederemos a la instalación de OwnCloud.

En la pagina principal ya nos solicitara que introduzcamos un usuario y una contraseña que usaremos como administrador, en la parte inferior nos solicitara los datos de la base de datos.

Configurar los datos

Solo tendríamos que darle a comenzar instalación y en cuestión de 1 minuto estaría instalada por completo.

Una vez completada la instalación, debemos introducir el nombre de usuario y la contraseña que hemos introducido anteriormente.

PrestaShop

Para la instalación de PrestaShop primero hemos tenido que descargarlo desde la Web oficial.


una vez descargado en el servidor vamos a extraerlo y le daremos ermisos de lectura y escritura a su carpeta para la instalación sin problemas.

comando

Una vez finalizado el anterior proceso accederemos a la dirección IP de nuestro servidor, ahí accederemos a la instalación de PrestaShop. Tendremos que esperar a que la pagina nos cargue mientras que extrae sus archivos.

Lo primero que nos solicita es que seleccionemos el idioma deseado, una vez seleccionemos el idioma nos aparece los términos y condiciones para aceptar.

Ahora el instalador detectara si tenemos todos los requisitos necesarios para poder realizar la instalación de PrestaShop, si todo esta bien nos dejara continuar con la instalación.

Llegamos al apartado donde rellenaremos los datos de la tienda: nombre, país y actividad de la tienda.

También en este apartado crearemos el usuario con cual administraremos la tienda.

Ahora solo queda esperar a se cree todas las bases de datos y finalice la instalación, esto tardara varios minutos.

WordPress

Para la instalación de WordPress primero hemos tenido que descargarlo desde la Web oficial

Una vez descargado en el servidor vamos a extraerlo y le daremos permisos de lectura y escritura a su carpeta para la instalación sin problemas.

comando

Una vez finalizado el anterior proceso accederemos a la dirección IP de nuestro servidor, ahí accederemos a la instalación de WordPress.

En la pagina rincipal nos indicara las pautas que vamos a realizar para instalación.

Empecemos con la primera pauta, nos pedirá los datos de la base de datos. Una vez rellenados hará una prueba de acceso a la base de datos y si fuera correcta nos dejara continuar.

Ahora llegamos a la ultima pauta que sería rellenar los datos del blog: nombre, usuario admin y su contraseña.

Tras finalizar de rellenar los datos daremos en instalar WordPress y al cabo de 1 minuto nos indicara que ya hemos finalizado la instalación.
